iso-work
========

Pisi Linux iso creation tools and files

Requirements
------------
 - xorriso
 - syslinux
 - squashfs-tools
 - pyqtermwidget5 (Required for GUI)
 - python-sip
 - python-qt5
 - python-qt5-devel
 - make
 - python-pathlib2
 - python-requests

Requirements kısımdaki programların sisteminizde kurulu olması gerekmektedir.
Çalıştırmadan önce pisiman/not dosyasını okuyun.
