Команда Anka з гордістю представляє Pisi Linux {release}
Pisi Linux - це дистрибутив операційної системи GNU/Linux із політикою оновлень "ролінг-реліз".
Його створено як нащадка дистрибутиву на основі PiSi - Pardus Linux, - після того, як його розробку було припенено у 2012 році.

Pisi Linux містить декілька унікальних технологій:
COMAR (COnfigurationMAnageR), налаштовує та інтегрує встановлене ПЗ.
MUDUR, прискорює процес завантаження.
PISI (Packages Installed Successfully, as Intended), пакетний менеджер, розроблений для Pardus Linux.
YALI (Yet Another Linux Installer), ручний установник системи.
KAPTAN, помічник із початкового налаштування робочого оточення.

Системні вимоги
64-розрядний процесор
Щонайменше 2 ГБ оперативної пам’яті та 12 ГБ вільного місця на диску.
Рекомендується створити розділ підкачки для Pisi Linux (доступне під час установлення)

Мультимедіа
Pisi Linux на даний момент є лише у 64-розрядній версії. Тим не менше, він містить 32-розрядні пакети для таких програм, як
Skype та Steam. Кодеки для відтворення аудіо- та відеофайлів також містяться в поставці.
А що ще?
У наших репозиторіях є понад 4000 пакетів і постійно додаються нові.

Pisi Linux перекладено 14 мовами: турецькою, англійською, голландською, німецькою, російською, іспанською, французькою, угорською,
італійською, хорватською, шведською, українською, польською та португальською (Бразилія). Відбувається переклад і іншими мовами.
